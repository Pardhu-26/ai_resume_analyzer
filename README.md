# 📄 AI Resume Analyzer & Job Match System

An intelligent web application that analyzes resumes and matches them with suitable job roles using Machine Learning and NLP techniques.

---

## 🚀 Features

* 📥 Upload resume (PDF format)
* 🧠 Extract and analyze resume content
* 📊 Job role matching using NLP (TF-IDF + Cosine Similarity)
* 📈 Visual score representation (progress bars + charts)
* 🎯 Top role recommendation
* ⚠️ Skill gap detection
* 💡 Smart suggestions to improve resume

---

## 🧠 Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas & NumPy
* PyMuPDF (PDF text extraction)

---

## 🏗️ How It Works

1. User uploads a resume (PDF)
2. Text is extracted from the resume
3. Resume content is compared with predefined job descriptions
4. TF-IDF vectorization + cosine similarity is used for matching
5. Scores are normalized and ranked
6. Best role is identified
7. Missing skills and suggestions are generated
